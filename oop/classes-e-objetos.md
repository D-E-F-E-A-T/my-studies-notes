# O que são classes e objetos em programação orientada a objetos

Uma classe é um modelo para objetos que contém métodos (funções) e propriedades. As classes não existem em memória, elas servem apenas para serem modelos de objetos. Classe Pessoa, Classe Fruta.
Já um objeto é uma classe que foi instanciada, que contém distintos dados armazenados em memória. Em outras palavras, um objeto é uma entidade que tem como modelo uma classe. Exemplo: Objeto Joao, Objeto Maria, Objeto Uva, Objeto Abacaxi.
